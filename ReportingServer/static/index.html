roqyBT problem reporting utility

Installation:

  • Add icrashrep.appspot.com source to Cydia
  • Install the 'roqyBT problem reporter' package

Usage:

  1. Press the 'Start logging' button, a checkbox should appear
  2. Leave the app, open roqyBT4 and try connecting to your device
  3. Open the reporting app again, click the Steps button, enter additional details (GPS model, any specific steps..); click Done when finished
  4. Click 'Send report' button (you need an active 3G or WiFi internet connection for that step); 'success' text with a check box should appear if upload was successful.
